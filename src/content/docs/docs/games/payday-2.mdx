## Loaders

Some PAYDAY 2 mods require SuperBLT or BeardLib. When Modrex can recognize a missing loader dependency, it shows a dependency warning and provides an install action or an external download path.

## Priority

PAYDAY 2 directory mods are not renamed for priority. Use Modrex folders, enable/disable controls, and the mod author's instructions to manage conflicts.
