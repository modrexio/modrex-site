PAYDAY 2 support covers BLT mods, BeardLib mods, asset replacement packs, and recognized content packs that install inside a host mod.

## Launchers

| Launcher   | Support |
| ---------- | ------- |
| Steam      | Yes     |
| Epic Games | Yes     |
| Xbox App   | No      |

## Mod targets

| Target                | Path under game folder     | Notes                                                                                                  |
| --------------------- | -------------------------- | ------------------------------------------------------------------------------------------------------ |
| BLT and BeardLib mods | `mods`                     | Modrex recognizes folders with `mod.txt` or `main.xml`.                                                |
| Asset replacements    | `assets/mod_overrides`     | Marker-less folders are routed here when they match the mod_overrides layout.                          |
| Host mod packs        | Inside the host mod folder | Some packs install inside another mod, such as Menu Backgrounds packs. Modrex tracks these separately. |

## Loaders

Some PAYDAY 2 mods require SuperBLT or BeardLib. When Modrex can recognize a missing loader dependency, it shows a dependency warning and provides an install action or an external download path.

## Priority

PAYDAY 2 directory mods are not renamed for priority. Use Modrex folders, enable/disable controls, and the mod author's instructions to manage conflicts.
