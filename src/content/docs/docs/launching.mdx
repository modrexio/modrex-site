## Launch modded

Click **Launch** in the top bar to start the active game with all enabled mods active. Modrex passes
your configured launch options to the game and hands off to your launcher.

For PAYDAY 3, make sure `-fileopenlog` is in your launch options - without it most `.pak` mods
won't load. See [PAYDAY 3](/docs/games/payday-3#launch-options) for details.

## Launch without mods

Click the dropdown arrow next to **Launch** and select **Launch without mods**. Modrex temporarily
backs up the active game's mod folders so the game starts without Modrex-managed mods.

When the game closes, Modrex automatically detects this and restores your mods.

See [Launch modes](/docs/concepts/launch-modes) for how target-scoped backups and restore behavior
work.

## Launcher support
