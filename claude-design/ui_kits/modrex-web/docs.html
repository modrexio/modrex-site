Installation

Modrex is a desktop app for Windows and Linux. Download the build for your platform, install it, and launch — there's no account to create and nothing to configure to get started.

Download & install

Grab the latest release from the download page or directly from GitHub.

Windows

Run Modrex_0.9.1_x64-setup.exe and follow the installer. Modrex checks for updates on launch and can update itself in-app.

Linux

Pick the package for your distribution:

  • modrex.deb — Debian, Ubuntu, and derivatives
  • modrex.rpm — Fedora, RHEL, and derivatives
  • modrex_0.9.1.AppImage — any distribution; mark it executable and run

First run

On first launch, Modrex automatically detects your PAYDAY 3 installation from your installed launchers. If it can't find the game, set the path manually:

  1. Open Settings from the sidebar.
  2. Under Game Path, click Browse and select your PAYDAY 3 folder.
  3. Modrex validates the folder and detects your launcher automatically.

Add -fileopenlog to your Launch Options in Settings. Most mods need it to load correctly.

Installing mods

Open Browse Mods, find a mod, and click Install. Modrex downloads it from modworkshop and places it correctly — supporting .pak, .zip, and .7z archives. Toggle a mod off to disable it without removing it, or organize your library into nested folders on the Installed tab.

Launching the game

Use Launch modded in the top bar to start PAYDAY 3 with your enabled mods, or Launch without mods for a clean vanilla session — Modrex hides your mods on the way out and restores them when the game closes.

Troubleshooting

If something goes wrong, open Settings → Logs → Open log file and attach it to a bug report. Log locations:

Windows   %APPDATA%\Modrex\logs\modrex.log
Linux     ~/.config/modrex/logs/modrex.log
